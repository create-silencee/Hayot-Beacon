#  Hayot Beacon (Маяк Жизни) — Безопасность и Помощь

**Проект разработан для Andijan Future Hackathon 2026** **Тема:** Система экстренного реагирования и первой помощи  
**Автор:** SamvelDev (@quiet_silencee)

---

## О проекте
**Hayot Beacon** — это интеллектуальная экосистема экстренной помощи. В критических ситуациях (ДТП, сердечный приступ, травма) время реакции — главный фактор спасения жизни. Наше решение объединяет пострадавших, профессиональных медиков-волонтеров и экстренные службы в одну сеть.

###  Ключевые функции (MVP):

1. **Мгновенный SOS:** Кнопка вызова помощи в один клик с автоматической передачей геопозиции.
2. **Функция «Я волонтер»:** Уникальная система уведомлений. Если кто-то поблизости нажал SOS, все зарегистрированные пользователи с медицинским образованием получают мгновенный сигнал, чтобы оказать помощь до приезда скорой.
3. ** Интерактивная карта тела:** Наглядный визуальный гид. Нажмите на область травмы — получите пошаговый алгоритм действий (остановка кровотечения, реанимация и т.д.).
4. ** Локатор ресурсов:** Карта ближайших дефибрилляторов, круглосуточных аптек и больниц города Андижан.
5. ** Дизайн "Спокойствие":** Современный интерфейс Apple Modern (Glassmorphism), созданный для минимизации паники пользователя.
